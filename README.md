# Bank Marketing Subscription Prediction

This repository contains a machine learning project for predicting whether a
bank customer will subscribe to a term deposit. The project uses the UCI Bank
Marketing dataset and implements a logistic regression pipeline from scratch,
including preprocessing, training, evaluation, threshold analysis, feature
analysis, and an interactive Streamlit dashboard.

## Repository Contents

| File | Description |
| --- | --- |
| `app.py` | Main Streamlit application. Run this file to open the interactive dashboard. |
| `MODEL1_FINAL_frontend.py` | Frontend-friendly version of the ML pipeline. It exposes reusable functions used by `app.py`. |
| `MODEL1_FINAL.py` | Original command-line ML pipeline script. It can be run directly to train and evaluate the model in the terminal. |
| `model 2.ipynb` | Jupyter notebook for additional model experimentation. |
| `bank-full.csv` | Local copy of the Bank Marketing dataset used by the Streamlit app. |
| `lr_model.pkl` | Serialized logistic regression model artifact generated by the pipeline. |
| `preprocessor.pkl` | Serialized preprocessing artifact generated by the pipeline. |
| `PAML_FINAL_PROJECT.pdf` | Final project report. |
| `README.md` | Project documentation and setup instructions. |

## Application Features

The Streamlit dashboard includes:

- Data exploration and class distribution summary
- Feature descriptions
- Preprocessing demonstration
- Logistic regression training results
- Evaluation metrics and confusion matrices
- Decision-threshold sensitivity analysis
- Feature coefficient analysis
- Single-customer subscription prediction form

## Setup Instructions

### 1. Clone or open the repository

Open a terminal in the project folder:

```bash
cd /path/to/nbnb-1
```

### 2. Create a virtual environment

```bash
python3 -m venv .venv
source .venv/bin/activate
```

On Windows, use:

```bash
python -m venv .venv
.venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install streamlit pandas numpy matplotlib ucimlrepo
```

The app uses the local `bank-full.csv` file by default. If that file is not
available, the pipeline can fall back to downloading the dataset through
`ucimlrepo`, which requires an internet connection.

## Run the Streamlit Application

From the repository root, run:

```bash
streamlit run app.py
```

Streamlit will start a local web server and print a URL similar to:

```text
Local URL: http://localhost:8501
```

Open that URL in a browser to use the dashboard.

## Run the Command-Line Pipeline

To run the original ML pipeline in the terminal:

```bash
python MODEL1_FINAL.py
```

This script loads the dataset, preprocesses the data, trains the logistic
regression model, evaluates performance, runs threshold and sensitivity
experiments, and saves the model artifacts.

## Notes

- The dataset is imbalanced, with a much smaller positive class for customers
  who subscribed to a term deposit.
- The Streamlit app trains the model during the session so the dashboard can
  display live pipeline results.
- `duration` is included for analysis but should be treated carefully in
  production settings because it is only known after a call has occurred.
